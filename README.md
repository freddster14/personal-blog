# personal-blog
Live Preview: [Click Here](https://personal-blog-sepia-iota.vercel.app/)

## What it is  
Full-stack blog application with client-server architecture.

## Features  
- User authentication
- Create, edit, delete posts
- Comments system  
- Role-based permissions (admin/user) 

## Tech Stack  
- Backend: Node.js, Express, Prisma, PostgreSQL
- Frontend: React
